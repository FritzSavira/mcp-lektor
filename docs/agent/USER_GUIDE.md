# Bedienungsanleitung: MCP-Lektor Agent

Herzlich willkommen! Dieser Agent hilft dir, deine Microsoft Word-Dokumente professionell zu lektorieren. Das Besondere: Dein Layout bleibt zu 100% erhalten, und alle Korrekturen werden als "Änderungen nachverfolgen" (Track Changes) sowie als Kommentare direkt in die Datei geschrieben.

## So startest du
1.  **Datei hochladen**: Lade deine `.docx`-Datei einfach in den Chat hoch.
2.  **Analyse abwarten**: Der Agent analysiert dein Dokument, zählt Wörter und findet Platzhalter (roter Text).
3.  **Korrekturen prüfen**: Der Agent präsentiert dir eine Liste mit Vorschlägen (Rechtschreibung, Grammatik, Typografie, Bibelstellen).

## Korrekturen steuern
Du hast die volle Kontrolle! Du kannst dem Agenten einfach sagen, was er tun soll:

- **"Alle akzeptieren"**: Der Agent übernimmt alle Vorschläge und erstellt direkt das neue Dokument.
- **"Nur Rechtschreibung"**: Nur die sichersten Fehler werden übernommen.
- **"C-001 akzeptieren, C-002 ablehnen"**: Du entscheidest für jeden Punkt einzeln (nutze dazu die IDs aus der Tabelle).
- **"Ändere C-003 in 'Beispieltext'"**: Du kannst den Vorschlag des Lektors durch deinen eigenen Text ersetzen.

## Tipps für beste Ergebnisse
- **Bibelstellen**: Der Agent erkennt Formate wie `Joh 3,16` oder `1. Mose 1,1`. Er warnt dich, wenn Kapitel- oder Versangaben nicht existieren.
- **Platzhalter**: Text in **roter Farbe** (z.B. `[Foto hier einfügen]`) wird vom Agenten bewusst ignoriert, damit deine redaktionellen Anmerkungen erhalten bleiben.
- **Anrede-Check**: Der Agent erkennt, ob du "Du" oder "Sie" verwendest und warnt dich bei Inkonsistenzen.

## Das Ergebnis
Nach Abschluss der Korrektur erhältst du einen Download-Link. Öffne die Datei in Word:
- **Rot markiert**: Gelöschter Text.
- **Unterstrichen**: Neuer Text.
- **Kommentare**: Erklärungen, warum eine Änderung vorgeschlagen wurde.
